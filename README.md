**💸 Penny Pilot**

A smart expense tracker app built with Node.js and Expo Go.Penny Pilot is a smart expense tracking and budgeting application designed to help users take control of their personal finances. It allows users to log daily expenses, categorize transactions, visualize spending patterns, and set savings goals.

**📖 Overview**

Penny Pilot is a mobile-first expense tracking application designed to help users manage their daily spending, set budgets, and gain insights into their financial habits. With a simple and intuitive interface, it enables users to log expenses on the go and visualize their spending through interactive charts.

***✨ Features***

➕ Add, edit, and delete expenses
🏷 Categorize transactions (food, travel, shopping, bills, etc.)
📊 Visualize spending with interactive charts
📅 Track weekly and monthly budgets
🔔 Get reminders when nearing budget limits
📱 Built with Expo Go for cross-platform mobile support

***🛠 Tech Stack***

Frontend: React Native (Expo Go)
Backend: Node.js + Express
Database: MongoDB / PostgreSQL (your choice)
Charts: Recharts / Victory Native


🚀 Getting Started
**🔹 Prerequisites**

Make sure you have installed:

1)Node.js

2)Expo Go
 app on your phone

npm or yarn package manager

🔹 Backend Setup
cd backend
npm install
npm start

🔹 Frontend Setup
cd frontend
npm install
npx expo start
Scan the QR code using Expo Go app to run the app on your mobile.
